HubSpot Setup Instructions

1. Create the Workflow

  • Go to Automation ‚Üí Workflows ‚Üí Create workflow
  • Choose "Contact-based"
  • Set enrollment trigger: Form submission = Ambassador Registration Form

2. Add Delays & Emails

  • Email 1: Send immediately
  • Delay: 3 days ‚Üí Email 2
  • Delay: 4 days ‚Üí Email 3 (Day 7 total)
  • Delay: 7 days ‚Üí Email 4 (Day 14 total)
  • Delay: 16 days ‚Üí Email 5 (Day 30 total)

3. Create Separate Quarterly Report Workflow

  • Create date-based workflow
  • Trigger on specific dates: March 17, June 17, Sept 17, Dec 17
  • Filter: Contact property "Ambassador Status" = "Active"

4. Personalization Tokens

Use these HubSpot tokens in your emails:

  • {{ contact.firstname }} - First name
  • {{ contact.ambassador_region }} - Custom property for region
  • {{ contact.ambassador_email }} - Their ambassador email

5. Track Engagement

  • Monitor email open rates (target: 40%+)
  • Track CTA clicks (especially resource downloads)
  • Flag contacts who don't reply to photos within 7 days
  • Set task reminder for team to follow up on low engagement
