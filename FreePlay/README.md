## Customization Guide

### Colors
Current brand gradient: `#667eea` (purple) to `#764ba2` (darker purple)

To change colors, search and replace:
- `#667eea` - Primary purple
- `#764ba2` - Secondary purple
- `#1a1a1a` - Dark text
- `#f8f9fa` - Light background sections

### Fonts
All designs use system fonts for fast loading:
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, sans-serif;
```

To use custom fonts, add Google Fonts link in `<head>`.

### Content
- Text content is embedded in HTML
- Easy to edit - just find and replace text
- All sections clearly labeled with comments
